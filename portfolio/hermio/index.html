HERMIO

A simple discord bot to make your time more enjoyable. Join the server to see the bot in action.

Github Repo - Github Repo of this project

Commands

Command Description
$inspire Genereate inspirational quote
$article Provides new interesting article from dev.to community
$bored Suggests a random activity
$food Displays random food image
$qr text Generates the QR Code for the text provided. If text contains spaces enclose it in double quotes
$help Dispalys help


Usage

If you want to extend the functionality of this clone the repo and create .env file in the project root directory and add your Discord BOT TOKEN

BOT_TOKEN=<YOUR_BOT_TOKEN>
