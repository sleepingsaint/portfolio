Preview

Web utility to get the metadata of the link using microlink api

Service

Web Link Preview, Javascript, React

Date

December 16, 2020

Challenge

Getting metadata for the url comes handy in many cases like thumbnails, link previews etc. But to extract the metadata from the url can be tedious some times. This project returns the metadata in json response format.

Solution

Microlink api does the heavy lifting. It scrapes the url and returns the metadata. It also caches the url so that if you want to requesting the metadata multiple times, you can get the data quickly.

How to use this?

enter the url that you want to get the metadata for and click the get preview button.

It will return the response (metadata) in json format.
